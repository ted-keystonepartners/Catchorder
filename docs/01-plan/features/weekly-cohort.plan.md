# Plan: 주간 코호트 잔존율 차트

## 1. 개요

### 1.1 기능명
주간 코호트 잔존율 표 (Weekly Retention Cohort Table)

### 1.2 목표
설치 완료된 매장이 주차별로 얼마나 잔존(이용)하고 있는지를 한눈에 파악할 수 있는 코호트 분석 표를 대시보드에 추가한다.

### 1.3 배경
- 현재 월별 설치 코호트(Sankey 차트)는 있으나, **주간 단위 잔존율 추적**이 불가능
- "12월 2주차에 설치된 매장이 현재까지 얼마나 이용중인가?" 질문에 답할 수 없음
- 매장의 장기 활성화 추이를 파악하여 이탈 방지 전략 수립 필요

## 3. 데이터 정의

### 3.1 입력 데이터

| 필드 | 타입 | 설명 | 소스 |
|------|------|------|------|
| `first_install_completed_at` | DateTime | 설치 완료일 | stores 테이블 |
| `orders` | Object | 날짜별 주문 건수 | `{ "2024-12-10": 5, ... }` |
| `store_id` | String | 매장 고유 ID | stores 테이블 |
| `store_name` | String | 매장명 | stores 테이블 |

### 3.2 출력 데이터 (코호트 테이블)

```
┌───────────┬───────────┬───────────┬───────────┬───────────┬───────────┐
│ 설치 주차  │  Week 0   │  Week 1   │  Week 2   │  Week 3   │  Week 4   │
├───────────┼───────────┼───────────┼───────────┼───────────┼───────────┤
│ 12월 1주  │ 15 (100%) │ 14 (93%)  │ 13 (87%)  │ 12 (80%)  │ 11 (73%)  │
│ 12월 2주  │ 20 (100%) │ 18 (90%)  │ 17 (85%)  │ 16 (80%)  │     -     │
│ 12월 3주  │ 12 (100%) │ 11 (92%)  │ 10 (83%)  │     -     │     -     │
│ 12월 4주  │  8 (100%) │  7 (88%)  │     -     │     -     │     -     │
└───────────┴───────────┴───────────┴───────────┴───────────┴───────────┘
```

### 3.3 잔존 정의

- **Week 0**: 해당 주차에 설치완료된 매장 수 (기준점, 항상 100%)
- **Week N (N≥1)**: 설치 후 N주차에 **주문이 1건 이상** 발생한 매장 수
- **잔존율**: `(Week N 매장수 / Week 0 매장수) × 100%`
- 주문 0건 = 미이용 = 잔존하지 않음
